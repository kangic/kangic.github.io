블로그를 잘 활용하거나 글을 많이 올렸던 것은 아니지만 텀블러에 계정이 있긴 있었다.
만들 당시 markdown 지원여부를 따지다보니 텀블러로 결정했었는데, 최근 글을 하나 작성하고 private으로 포스팅하려다 보니 cpp code를 넣는데에서 상당히 애를 먹었다.(내가 잘 몰라서 그런걸지도..)

그래서 해결책을 이것저것 찾다가 github.io에 jekyll 환경으로 구축하면 markdown 지원은 물론 github repo로 글을 올리면 되니 굉장히 간편한 방법으로 관리할 수 있겠다 싶어서 변경하기로 마음 먹었다.
이것저것 조사하면서 세팅을 하고 블로그에 있던 몇 안되는 글을 옮겨보니 굉장히 만족스러웠고, 무엇보다 편집과 동시에 글을 올리는 과정도 굉장히 심플해져서(git add / commit / push) 좋다!

개발자에게는 이만한 환경이 없는 듯!
결론은 github 짱!!ㅎㅎ
